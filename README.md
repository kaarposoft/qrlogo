QR-Logo
=======

<http://qrlogo.kaarposoft.dk>

QR-Logo can embed your logo into a
<a href="http://en.wikipedia.org/wiki/QRcode">QR Code</a>.

QR-Logo is pure JavaScript, so it runs in the browser.
